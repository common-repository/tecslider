Slider Blocks to showcase your events.

== Description ==

Display your upcoming or past events with this Gutenberg Block. Easy to choose between your past or your upcoming events. You just need to insert the block, elementor widget or shortcode and your events will be ready to display in beautiful slider.

**Gutenberg Block**

* Select upcoming or past event(s)
* Select different themes.
* Select number of event(s)
* Select color of your choice with help of color picker.
* Change number of slides to show
* Change number of slides to scroll

**Elementor Widget**
* Search for 'slider', 'calendar', 'events', 'tecslider', 'glide' and look for 'EventGlide' in General section.

**Plugin Shortcode**

`[add_mwt_tec_slider slidertheme="datetop" slidertype="upcoming" eventinrow="3" eventinslide="2" autoplay=true transitiondelay="100" colorscheme="#215455" newtab=true category="party"]`

**slidertheme**
&nbsp;&nbsp; datetop
&nbsp;&nbsp; coloredcard
&nbsp;&nbsp; singleevent

**slidertype**
&nbsp;&nbsp; upcoming
&nbsp;&nbsp; past

**eventinrow**
&nbsp;&nbsp; Number of event(s) in a row

**eventsinslide**
&nbsp;&nbsp; Number of event(s) to slide

**autoplay**
&nbsp;&nbsp; true
&nbsp;&nbsp; false

**transitiondelay**
&nbsp;&nbsp; Transition delay in milliseconds ( Default: 200 )

**colorscheme**
&nbsp;&nbsp; Hex color ( Default: #5a30f3 )

**newtab**
&nbsp;&nbsp; Open event in new tab on click, true or false ( Default: true )

**category**
&nbsp;&nbsp; Filter by category ( use category slug )

**slideheight**
&nbsp;&nbsp; Changes to slider height (description part) wherever applicable (Default:300)

**imageheight**
&nbsp;&nbsp; Changes to image height wherever applicable (Default: 200)

**ticketinfo**
&nbsp;&nbsp; true
&nbsp;&nbsp; false
